# Focus
Web Scraper designed to crawl and collect course information from California community colleges. Parses data into a .csv in order to keep data well-organized and formatted. Implemented in Python.

# Depenedencies:

**Drivers:**
- [Chrome Drivers](https://googlechromelabs.github.io/chrome-for-testing/)
- [Firefox Drivers](https://github.com/mozilla/geckodriver/releases)

**Python Modules:**
- [Selenium](https://pypi.org/project/selenium/)
- [Selenium-Wire](https://pypi.org/project/selenium-wire/)
- [Blinker 1.7.0](https://pypi.org/project/blinker/)
